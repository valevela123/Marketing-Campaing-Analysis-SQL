Análisis de Campañas META vs TikTok ADS

Este proyecto simula un entorno real de Marketing Analytics donde se analizan datos de inversión y ventas.

Herramientas utilizadas:
SQL Server: Estructuración de base de datos y consultas complejas.
Power BI: Proceso de ETL y diseño de Dashboard interactivo.
DAX: Creación de medidas para ROI y CTR.

Hallazgos principales:
Meta Ads presentó un ROI de 7.0 frente al 2.2 de TikTok.
Se identificó que TikTok tiene un CTR más alto, sugiriendo una buena etapa de awareness pero baja conversión final comparada con Meta.
